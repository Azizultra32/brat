## Quick Command Reference

| Command | Description |
|---------|-------------|
| `brat init` | Initialize harness |
| `brat status` | View status |
| `brat convoy create` | Create convoy |
| `brat convoy list` | List convoys |
| `brat task add` | Add task |
| `brat task list` | List tasks |
| `brat mayor start` | Start Mayor |
| `brat mayor ask` | Ask Mayor |
| `brat witness run` | Run Witness |
| `brat refinery run` | Run Refinery |
| `brat daemon start` | Start daemon |
| `brat daemon status` | Check daemon |

See [CLI Reference](cli.md) for complete details.

## Global Flags

Available on all commands:

| Flag | Description |
|------|-------------|
| `--json` | JSON output |
| `--quiet` | Suppress output |
| `--repo <path>` | Target specific repo |
| `--no-daemon` | Don't auto-start daemon |
| `--verbose` | Verbose output |
| `--help` | Show help |
| `--version` | Show version |
